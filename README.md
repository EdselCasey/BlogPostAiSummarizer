# AI Blog Summarizer

Paste any blog article URL and receive a concise AI-generated summary.

---

## Features

- Extracts article content automatically from a provided URL
- AI-powered summarization
- Clean modern UI built with TailwindCSS
- Instant results with a simple interface
- GitHub button linking back to the source repository

---

## Tech Stack

- **Frontend:** React
- **Styling:** TailwindCSS
- **AI Integration:** AI API for summarization
- **Networking:** Fetch API / Async JavaScript

---

## How It Works

1. User submits a blog article URL.
2. The application fetches and extracts the main article content.
3. The extracted text is sent to an AI model.
4. The AI generates a concise summary of the article.
5. The summary is returned and displayed in the UI.

---

## Use Case

This tool helps users quickly understand long-form articles without needing to read the entire post. It is useful for:

- Researchers
- Students
- Professionals
- Anyone who wants a quick overview of online content
